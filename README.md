# Skilled Language

Searchable library of audit-defensible PT, OT, and ST documentation phrases for Medicare Part B compliance.

## Features

- 275+ phrases across 17 clinical categories
- Filter by discipline (PT, OT, ST) or browse all
- Full-text search
- Save favorites with persistent storage
- One-click copy on every phrase
- Single HTML file, no dependencies
